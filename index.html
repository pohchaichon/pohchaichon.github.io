ARMY

We are exploring various database components on ARM processors. Major cloud vendors has been offering more ARM-based servers due to its cost efficiency, and to that end, we hope to accelerate the adoption of ARM-based cloud. With its unique characteristics (weak-ordered memory model, NUMA, large core counts, etc.) compared to x86 processors, we aim to study the implications and considerations when implementing a database system on ARM-based server.

Specifically:

  • Relational operators (e.g., Hash and Sort joins)
  • Indexing
  • Transactional Concurrency Control
  • Locking Primitives
